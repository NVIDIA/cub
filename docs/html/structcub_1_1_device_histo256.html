cub::DeviceHisto256 Struct Reference
Device-wide

Detailed description

DeviceHisto256 provides device-wide parallel operations for constructing 256-bin histogram(s) over samples data residing within global memory.

.

Computes a 256-bin device-wide histogram. Uses fast block-sorting to compute the histogram.

Delivers consistent throughput regardless of sample diversity.

Template Parameters
InputIteratorRA[inferred] The random-access iterator type for input (may be a simple pointer type). Must have a value type that is assignable to unsigned char
HistoCounter[inferred] Integral type for counting sample occurrences per histogram bin
Parameters
