Detailed description

template<typename T>
class cub::TexIteratorRA< T >

A simple random-access iterator for loading primitive values through texture cache.

Overview
TexIteratorRA is a random-access iterator that wraps a native device pointer of type T*. References made through TexIteratorRA causes values to be pulled through texture cache.
Usage Considerations
  • Can only be used with primitive types (e.g., char, int, float), with the exception of double
  • Only one TexIteratorRA or TexIteratorRA of a certain InputType can be bound at any given time (per host thread)
