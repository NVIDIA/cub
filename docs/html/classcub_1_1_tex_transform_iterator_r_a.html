template< typename OutputType, typename ConversionOp, typename InputType>
class cub::TexTransformIteratorRA< OutputType, ConversionOp, InputType >

A simple random-access transform iterator for loading primitive values through texture cache and and subsequently applying a transformation operator.

Overview
TexTransformIteratorRA is a random-access iterator that wraps both a native device pointer of type InputType* and a unary conversion functor of type ConversionOp. OutputType references are made by pulling InputType values through the texture cache and then transformed them using the ConversionOp instance.
Usage Considerations
  • Can only be used with primitive types (e.g., char, int, float), with the exception of double
  • Only one TexIteratorRA or TexTransformIteratorRA of a certain InputType can be bound at any given time (per host thread)
Template Parameters
InputTypeThe value type of the pointer being wrapped
ConversionOpUnary functor type for mapping objects of type InputType to type OutputType. Must have member OutputType operator()(const InputType &datum).
