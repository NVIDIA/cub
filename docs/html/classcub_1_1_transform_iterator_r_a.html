template< typename OutputType, typename ConversionOp, typename InputType>
class cub::TransformInputIterator< OutputType, ConversionOp, InputType >

A simple random-access transform iterator for applying a transformation operator.

Overview
TransformInputIterator is a random-access iterator that wraps both a native device pointer of type InputType* and a unary conversion functor of type ConversionOp. OutputType references are made by pulling InputType values through the ConversionOp instance.
Template Parameters
InputTypeThe value type of the pointer being wrapped
ConversionOpUnary functor type for mapping objects of type InputType to type OutputType. Must have member OutputType operator()(const InputType &datum).
OutputTypeThe value type of this iterator
