template< typename ReductionOp, typename ItemOffsetPair>
class cub::ReduceBySegmentOp< ReductionOp, ItemOffsetPair >

Reduce-by-segment functor.

Given two cub::ItemOffsetPair inputs a and b and a binary associative combining operator f(const T &x, const T &y), an instance of this functor returns a cub::ItemOffsetPair whose offset field is a.offset + a.offset, and whose value field is either b.value if b.offset is non-zero, or f(a.value, b.value) otherwise.

ReduceBySegmentOp is an associative, non-commutative binary combining operator for input sequences of cub::ItemOffsetPair pairings. Such sequences are typically used to represent a segmented set of values to be reduced and a corresponding set of {0,1}-valued integer "head flags" demarcating the first value of each segment.< ItemOffsetPair pairing of T (value) and OffsetT (head flag)
